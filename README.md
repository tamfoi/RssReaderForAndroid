# RssReaderForAndroid

# 使い方

## RSS の登録方法

- トップ画面でオプションメニューから「RSS を追加/削除」を選択します。
- RSS の URL を入力します。
- 「追加する」ボタンを押します。

## RSS の削除方法

- トップ画面でオプションメニューから「RSS を追加/削除」を選択します。
- URL リストの消したい URL を長押しします。
- 消すかどうか質問されますので「OK」を押します。

## RSS から記事リストを取得する方法

- トップ画面でスピナーをタップします。
- 登録されている RSS リストが表示されるので、見たい RSS を選択します。
- 記事リストを取得して表示されます。
